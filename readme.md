# 🛡️ Log Analyzer (SOC Project)

## 📌 Overview
This project is a **Log Analyzer built using Python** that simulates real-world SOC (Security Operations Center) activities. It scans log files to detect suspicious behavior such as failed login attempts, brute-force attacks, and potential SQL injection patterns.

---

## 🎯 Objective
To automate the process of analyzing system logs and identifying potential security threats using Python scripting.

---

## ⚙️ Features
- 🔍 Detects multiple failed login attempts (401 errors)
- 🚨 Identifies brute-force attack patterns
- 💀 Detects basic SQL Injection attempts
- 🔐 Flags access to sensitive paths (e.g., `/admin`)
- 📊 Generates alert outputs for suspicious activity
- ⚡ Fast and automated log processing

---

## 🧠 Concepts Covered
- Log Analysis (SOC Fundamentals)
- Cybersecurity Threat Detection
- Python Automation & Scripting
- Pattern Matching & String Searching
- Regular Expressions (Regex)
- Basic Incident Detection
